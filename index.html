Climate Latitude Visualizations

This site showcases weather data visualizations summarizing more than 500 random cities to help answer the question of how weather changes as you approach the equator. This is primarily an analysis of the relation of latitude to four different weather parameters, namely temperature, humidity, cloudiness, and wind speed. The data gathered for this analysis is from Jan 5th, 2017, using the free version of the Open Weather Map API for current weather data.
